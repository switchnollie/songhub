Uploads a file with the name fileName to the filePath in Firebase Storage and returns the upload path.

If isPublic is set, the file will be uploaded inside the public folder. If not, meta data describing the permissions is created using userPermissions.

Implementation

Future<String> uploadFile(
    {String filePath,
    File file,
    String fileName,
    bool isPublic = false,
    FileUserPermissions userPermissions}) async {
  StorageReference ref;

  if (isPublic) {
    ref = _storage.ref().child("public/$filePath/$fileName");
  } else {
    ref =
        _storage.ref().child("${userPermissions.owner}/$filePath/$fileName");
  }

  StorageMetadata meta = userPermissions != null
      ? createMetadata(userPermissions.owner, userPermissions.participants)
      : null;

  StorageUploadTask uploadTask = ref.putFile(file, meta);

  await uploadTask.onComplete;

  return await ref.getPath();
}
