SongDetailsViewModel class

A view model to handle all song details data.

This view model requires song specific data as well as service instances for Firebase Firestore, Storage and Authentication. Getters are defined to receive events specific to data used in this view. Streams are initialized for Song, Recordings, Messages. When fetching data, further functionality is used to fetch Storage specific data. All data is mapped into view models.
