Sends a sign in request to Firebase Authentication using the credentials email and password.

If successful, it signs the user in, updates the onAuthStateChanged stream and returns a FireUser.

Implementation

Future<FireUser> signInWithEmailAndPassword(
    String email, String password) async {
  final AuthResult authResult = await _firebaseAuth
      .signInWithCredential(EmailAuthProvider.getCredential(
    email: email,
    password: password,
  ));
  return _userFromFirebase(authResult.user);
}
